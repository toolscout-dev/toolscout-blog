# Lessons Learned - ToolScout

## SEO Insights
- Open source alternatives are a strong niche—lower competition, engaged audience
- Pricing comparison angles ("X% cheaper than") work well for click-through
- GitHub trending repos make great content opportunities

## Content Patterns
- Structure: Hook → What is it → Features → Pricing → Verdict
- Include comparison tables for pricing—they're highly scannable
- Code snippets (self-hosting commands) add value for technical readers

## Mistakes to Avoid
- Don't use && in PowerShell commands—use ; instead
- Always check if tracking files exist before reading
